# PRD: 2026 世界杯球迷互动站

## 版本
v0.1 - 2026-06-07

## 概述
面向中国球迷的 2026 FIFA 世界杯娱乐互动网站，提供赛程比分、竞猜、球星卡收集、排行、社区讨论等功能，以提升用户参与度和粘性为核心目标，非盈利性质。

## 目标用户
- 中文世界杯球迷（18-40岁）
- 有竞猜/互动需求的活跃球迷
- 边看球边刷手机聊球的群体

## 技术选型（初定）
- **前端**: HTML + CSS + JavaScript（纯静态网站，快速上线验证）
- **数据**: 世界杯赛程/比分数据通过公开 API 获取
- **部署**: 待定

## 功能列表

### P0（MVP 必做）
1. **赛程 & 实时比分**
   - 展示当日/全部赛程
   - 比赛时间、对阵双方、分组信息
   - 实时比分更新（进球动态）
   - 球队国旗/徽章展示
   - *优先级：P0*

2. **每日竞猜**
   - 用户可预测比赛胜负/比分
   - 猜对计分，猜错不扣分
   - 连胜奖励机制（连续猜中 streak 加成）
   - 显示竞猜结果和统计
   - *优先级：P0*

3. **个人积分排行榜**
   - 按竞猜得分排名
   - 周榜 / 总榜切换
   - 显示排名、用户名、积分、头衔
   - *优先级：P0*

4. **球星卡收集**
   - 通过竞猜/签到获得抽卡机会
   - 开启卡包获得随机球员卡
   - 不同稀有度：普通 / 稀有 / 传奇
   - 收集进度显示（已收集 / 总数）
   - *优先级：P0*

5. **社区讨论区**
   - 用户发帖讨论比赛/球员/卡牌
   - 点赞、评论互动
   - 帖子流按时间排序
   - 支持表情/梗图氛围
   - *优先级：P0*

### P1（后续迭代）
- 好友 PK 竞猜
- 每日签到 + 连续打卡
- 每日任务系统
- 最佳阵容评选（TOTD）

### P2（远期规划）
- Fantasy 梦幻足球
- 实时推送通知
- 微信小程序版

## 非功能性需求
- 移动端适配（大部分用户手机浏览）
- 页面加载快，轻量化
- 配色统一，品牌感强
- 无用户登录系统（初期简化）
