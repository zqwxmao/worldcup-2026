# Design Spec: 2026 世界杯球迷互动站

## 版本
v0.1 - 2026-06-07

## 整体风格
**绿茵竞技场**（来源：方案B活力竞技场 + FIFA图片草绿色系提取）

风格关键词：清新、活力、运动、绿茵场

## 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色（草绿） | `#1B8A4A` | 导航/按钮/标题，FIFA 球场绿提取 |
| 亮色（鲜绿） | `#00C853` | hover/选中态/accent |
| 深色（深林） | `#0D4A2B` | Hero 渐变色/深色元素 |
| 辅色（FIFA蓝） | `#1A4A8A` | 辅助色/链接色 |
| 金色（排行） | `#F5A623` | 排行榜/星标/传奇卡 |
| 橙色（CTA） | `#FF6D00` | 主要操作按钮 |
| 背景 | `#F4F7F4` | 页面底色，带绿色调的白 |
| 卡片 | `#FFFFFF` | 纯白卡片 |
| 深色文字 | `#1A2A1A` | 正文 |
| 浅色文字 | `#5A7A6A` | 辅助文字 |
| 边框 | `#D8E8D8` | 卡片/分割线 |

## 字体
- 系统字体栈：`-apple-system, "PingFang SC", "Segoe UI", Roboto, sans-serif`
- 标题：粗体 700-800
- 正文：常规 400-500
- 数字：加粗，突出竞猜/排行数据

## 页面清单

### 1. 首页（MVP）
- 顶部导航栏（Logo + 导航链接 + 用户积分头像）
- Hero 横幅（倒计时 + 开幕信息 + CTA 按钮）
- 今日比赛卡片（3列，显示对阵/时间/预测按钮）
- 球星卡收集区（展示4张卡片 + 开包按钮）
- 排行榜区（本周Top5）
- 社区热帖区（2条帖子预览）
- 页脚

### 2. 赛程页（P1）
- 全部比赛列表
- 按日期/分组筛选
- 实时比分更新

### 3. 竞猜页（P1）
- 可选比赛列表
- 比分/胜负预测表单
- 竞猜记录

### 4. 卡牌图鉴页（P1）
- 全部卡牌网格展示
- 收集进度条
- 开启动效

### 5. 排行榜页（P1）
- 完整排行榜
- 周榜/月榜/总榜切换

### 6. 社区页（P1）
- 帖子列表
- 发帖功能
- 互动按钮

## 设计参考
- FIFA 官网（配色灵感）
- Goal.com（卡片式布局）
- 懂球帝（中文社区氛围）

## 交互风格
- 按钮 hover 有上浮效果 + 阴影
- 卡片 hover 有边框变色提示
- 卡牌 hover 上浮 + 轻微旋转
- 平滑过渡，不过度动画
